O objetivo é garantir que todos os alunos, independentemente de suas
diferenças individuais, tenham igualdade de acesso a oportunidades
educacionais de qualidade, respeito, desenvolvimento de uma cultura
inclusiva esse beneficiem plenamente do sistema educacional. A
acessibilidade nas escolas é uma questão fundamental para garantir que
todos os estudantes tenham oportunidades iguais de aprendizado e
desenvolvimento. Quando uma escola é acessível, ela permite que
todos os alunos participem igualmente das atividades escolares.
